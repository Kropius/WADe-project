1. Description

    The application Rest Api Interactive Tool(known as RAT) is a web tool that uses natural language constructs to interact with different REST APIs, specifically their OpenApi documentation. Based on this, a human-like(text based/voice) interaction with the given APIs will be possible. The end user will have the possibility to upload a valid OpenApi document, so the user will be able to interact in a natural manner with the API using the web interface of the application.

    The application will be able to determine valid routes, with valid request bodies or even query parameters based on the user input according to the uploaded/selected RestApi.
